# Projeto: Cronômetro Digital com Registro de Histórico

## Descrição
Este projeto consiste em um cronômetro funcional desenvolvido como treino para a linguagem de JavaScript. A aplicação permite a contagem progressiva de tempo (horas, minutos e segundos) e oferece a funcionalidade de registrar marcas temporais específicas durante o uso.

## Tecnologias Utilizadas
- HTML5: Estruturação dos elementos da interface.
- CSS3: Estilização com foco em design responsivo.
- JavaScript (ES6+): Lógica de manipulação de tempo, gerenciamento de intervalos e manipulação dinâmica do DOM.

## Funcionalidades
- Iniciar contagem: Ativa o cronômetro com precisão de segundos.
- Pausar contagem: Interrompe a progressão do tempo mantendo o valor atual.
- Reiniciar: Zera todos os contadores e interrompe processos ativos.
- Gravar Tempo: Registra o momento atual em uma lista de histórico perto do visor principal.
